## Descripción y justificación de la temática
Elegí crear un sitio sobre **mis libros favoritos** porque amo leer y quería compartir recomendaciones.  
Las **tarjetas reutilizables** son perfectas para este contenido: cada libro tiene la misma estructura (imagen, título, autor, reseña), pero algunas se destacan (mi favorito absoluto).  
El uso de **clases** (.tarjeta, .tarjeta--destacada) me permitió reutilizar el mismo diseño en todas las páginas, mientras que los **id** (#titulo-principal, #contacto) identifican elementos únicos y permiten navegación interna suave.

## Tecnologías usadas
- HTML5 semántico
- CSS3 (Flex, Grid, ::after, scroll-behavior)
- Google Fonts: Playfair Display + Open Sans
